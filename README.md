# Currency Converter

A responsive currency converter built using HTML, CSS, and JavaScript with local data.

## Description

This project provides a simple yet effective currency converter that works with local data. It is fully responsive and works across various devices.

## Features

- **Responsive Design**: Adapts to different screen sizes for a seamless experience.
- **Local Data**: Converts currencies using predefined exchange rates stored locally.
- **User-Friendly Interface**: Simple and intuitive design for easy conversion.
- **Cross-Browser Compatibility**: Works well on modern browsers.
